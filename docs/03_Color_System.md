# 03 — Color System

## Core Palette

| Color Name | HEX | Primary Use |
|---|---|---|
| NorthWing Midnight | `#051231` | Hero backgrounds, footer, dark sections |
| NorthWing Navy | `#052862` | Headings, navigation, primary brand surfaces |
| NorthWing Blue | `#033F91` | Cards, interface accents, secondary brand elements |
| Flight Blue | `#0D69CE` | Links, hover states, active UI, digital highlights |
| Flight Gold | `#EDB743` | CTAs, directional accents, key highlights |
| NorthWing Slate | `#5D6981` | Secondary text, subdued UI elements |
| Steel Gray | `#95979C` | Captions, dividers, tertiary labels |
| Cloud Gray | `#CED3DB` | Borders, light backgrounds, disabled elements |
| Wing White | `#FFFFFF` | Main light backgrounds, reversed text |

## Recommended Color Roles

### NorthWing Midnight
Use for:
- Hero backgrounds
- Footer
- Dark website sections
- Presentation covers
- Premium technology backgrounds

### NorthWing Navy
Use for:
- Headings
- Navigation
- Dark buttons on light backgrounds
- Document headers
- Iconography

### NorthWing Blue
Use for:
- Secondary buttons
- Interface elements
- Card accents
- Diagrams
- Section highlights

### Flight Blue
Use for:
- Links
- Hover states
- Active controls
- Workflow nodes
- Digital highlights

### Flight Gold
Use selectively for:
- Primary CTA buttons
- Directional arrows
- Small icon accents
- Key metrics
- Eyebrow labels
- Flight-path graphics

## Recommended Balance

Approximate visual distribution:

- 45% Wing White or light neutrals
- 25% NorthWing Midnight / Navy
- 15% Blue family
- 10% Gray neutrals
- 5% Flight Gold

## Approved Gradients

| Gradient | Values | Use |
|---|---|---|
| Primary Brand Gradient | `#051231 → #052862` | Hero, footer, presentation covers |
| Digital Blue Gradient | `#033F91 → #0D69CE` | Icons, UI highlights, graphic overlays |
| Flight Gold Accent | `#EDB743 → brighter warm gold` | Small directional and illustration accents |

## Accessibility Guidance

- Use Wing White text on NorthWing Midnight and NorthWing Navy.
- Use NorthWing Midnight for body text on white backgrounds.
- Avoid Steel Gray for important small body text.
- Reserve Flight Gold for emphasis rather than long-form text.
