# 08 — Voice and Tone

## Voice Principles

NorthWing Labs should sound:

- Clear
- Practical
- Confident
- Helpful
- Direct
- Technically credible
- Business-focused

## Communication Style

Explain the business outcome first, then the technology.

Preferred:

> We automate repetitive work so your team can spend more time serving customers and growing the business.

Avoid:

> We leverage advanced AI-powered orchestration frameworks to transform next-generation workflows.

## Core Messaging Pattern

Use this sequence:

**Problem → Solution → Business Outcome**

Example:

> Manual quoting slows down your response time. We can build a streamlined quoting workflow that reduces repetitive work and helps you respond to customers faster.

## Word Choice

Prefer:

- Build
- Simplify
- Automate
- Connect
- Improve
- Save time
- Reduce manual work
- Make easier
- Grow
- Practical
- Useful

Use technical terminology only when it adds necessary precision.

## Tone by Context

| Context | Tone |
|---|---|
| Website | Clear, confident, approachable |
| Proposal | Precise, practical, professional |
| Social Media | Educational, direct, helpful |
| Technical Documentation | Exact, structured, unambiguous |
| Sales Conversation | Consultative, problem-focused, low-pressure |

## Brand Message Test

Every major piece of copy should answer at least one of these questions:

- What business problem does this solve?
- What becomes easier or faster?
- What manual work is reduced?
- What improves for the customer?
- What result should the business expect?
