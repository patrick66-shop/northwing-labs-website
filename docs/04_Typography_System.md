# 04 — Typography System

## Primary Font Pairing

| Role | Font | Recommended Weight | Use |
|---|---|---:|---|
| Hero Headline | Manrope | 700–800 | Main website hero statement |
| H1 | Manrope | 700 | Primary page titles |
| H2 | Manrope | 700 | Major section headings |
| H3 | Manrope | 600 | Card and subsection headings |
| Eyebrow Label | Manrope | 700 | Small uppercase section labels |
| Body Large | Inter | 400 | Introductory paragraphs |
| Body Standard | Inter | 400 | General website copy |
| Body Emphasis | Inter | 600 | Important phrases and lead-ins |
| Button Label | Inter | 600 | CTA and interface buttons |
| Navigation | Inter | 500–600 | Header and footer navigation |
| Caption / Metadata | Inter | 400–500 | Captions, dates, support text |
| Software UI | Inter | 400–600 | Forms, dashboards, apps, and SaaS interfaces |

| Body | 17–18px | 1.60 |

## Eyebrow Label Style

| Property | Recommendation |
|---|---|
| Font | Manrope |
| Weight | 700 |
| Size | 13–14px |
| Case | Uppercase |
| Letter Spacing | Approximately `0.08em` |
| Color | Flight Blue or Flight Gold |

## Typography Rules

- Use Manrope for branded expression and hierarchy.
- Use Inter for readability and interface clarity.
- Keep the NORTHWING wordmark typography exclusive to the logo.
- Do not use the logo type style as a website heading font.
- Maintain generous line spacing.
- Avoid visually compressed body copy.

## Typography Principle

**Logo wordmark = distinctive identity**  
**Manrope = brand voice**  
**Inter = functional clarity**
