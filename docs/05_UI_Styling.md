# 05 — UI Styling

## Primary Button

| Property | Recommendation |
|---|---|
| Background | Flight Gold `#EDB743` |
| Text | NorthWing Midnight `#051231` |
| Font | Inter 600 |
| Shape | Medium corner radius |
| Use | Primary conversion action |

Example labels:

- Start a Conversation
- Book a Discovery Call
- Discuss Your Project
- Get Started

## Secondary Button

### On Dark Backgrounds

| Property | Recommendation |
|---|---|
| Background | Transparent |
| Border | Wing White |
| Text | Wing White |

### On Light Backgrounds

| Property | Recommendation |
|---|---|
| Background | NorthWing Navy |
| Border | NorthWing Navy |
| Text | Wing White |

## Links

- Default: Flight Blue
- Hover: NorthWing Blue
- Underline links inside long-form content
- Do not use Flight Gold as the default link color

## Cards

Recommended default card:

- Wing White background
- Cloud Gray border
- Subtle shadow
- NorthWing Navy heading
- Inter body copy
- Blue or gold accent used sparingly

## Form Fields

- White or very light neutral background
- Cloud Gray border
- NorthWing Midnight input text
- NorthWing Slate helper text
- Flight Blue focus state
- Clear labels above fields

## Section Labels

Use Manrope 700, uppercase, with moderate letter spacing.

Approved accent colors:

- Flight Blue
- Flight Gold

## Design Principle

UI elements should feel practical, clean, and product-oriented. Avoid decorative complexity that reduces clarity or resembles generic AI-generated interface design.
