# 02 — Logo System

## Approved Logo Family

The NorthWing Labs logo system includes:

| Logo Version | Recommended Use |
|---|---|
| Primary Logo | General brand use and large-format applications |
| Horizontal Logo | Website header, navigation, email signature, documents |
| Reversed Logo | Dark navy backgrounds, footer, dark presentation slides |
| Monochrome Logo | Single-color printing, stamps, low-color applications |
| Logo Mark | Favicon, app icon, avatar, compact brand placement |
| Square Profile Logo | Facebook, LinkedIn, Google Business Profile, social avatars |

## Logo Mark

The logo mark uses:

- Geometric blue N structure
- Upward-moving Flight Gold accent
- Angular, directional geometry
- Visual association with motion, growth, technology, and flight

## Usage Principles

- Keep the logo visually separated from surrounding content.
- Do not stretch, compress, rotate, skew, or redraw the logo.
- Do not recolor individual logo elements outside the approved palette.
- Do not add shadows, bevels, outlines, glows, or decorative effects.
- Do not place the primary dark logo on dark backgrounds.
- Use the reversed logo on NorthWing Midnight and NorthWing Navy backgrounds.
- Use the logo mark only where the full brand name is already known or space is limited.

## Clear Space

Maintain clear space around the logo equal to at least the approximate width of the inner vertical stroke of the N mark.

For compact digital placement, preserve enough separation so the mark and wordmark remain visually distinct from navigation, buttons, or other logos.

## Minimum Size Guidance

| Application | Minimum Recommendation |
|---|---|
| Horizontal Logo — Web | 140px wide |
| Primary Logo — Print | 1.5 in wide |
| Logo Mark — Web | 24px |
| Social Profile Image | 320×320px minimum source |
| Favicon Master | 512×512px source |

## Incorrect Usage

Do not:

- Place the logo over visually busy imagery without a clean overlay
- Use unapproved gradients
- Change Flight Gold to another accent color
- Separate the gold accent from the N mark
- Replace the wordmark typography with a similar font
- Rebuild the logo using normal text
