# 06 — Iconography

## Core Service Icon Set

Create a coordinated icon family for:

1. Custom Software
2. AI Tools
3. Business Automation
4. Mobile Apps
5. SaaS Development

## Icon Style

| Attribute | Standard |
|---|---|
| Geometry | Clean, geometric |
| Stroke | Thin to medium |
| Primary Colors | NorthWing Blue and Flight Blue |
| Accent | Small Flight Gold highlight |
| Corners | Rounded where appropriate |
| Detail Level | Minimal |
| Perspective | Consistent across the full set |

## Icon Usage

Use icons for:

- Service cards
- Process steps
- Feature highlights
- Proposal sections
- Social posts
- Presentation slides
- Software concept diagrams

## Avoid

- Mixing outline and filled icon families
- Random multicolor stock icons
- Cartoon illustration styles
- Excessive gradients
- Inconsistent stroke thickness
- Different perspectives across a single icon family

## Visual Principle

The icon system should feel like one engineered family, not five unrelated graphics.
