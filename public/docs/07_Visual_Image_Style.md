# 07 — Visual and Image Style

## Visual Direction

NorthWing Labs imagery should communicate practical technology solving real business problems.

The brand should avoid:

- Generic glowing AI brains
- Anonymous stock teams pointing at charts
- Overly futuristic neon cyberpunk scenes
- Random code screens used as decoration
- Visual complexity without business meaning

## Preferred Visual Themes

Use imagery that shows:

- Business workflows
- Connected systems
- Process improvement
- Software dashboards
- Mobile application interfaces
- Automation flows
- AI assistants
- Operational visibility
- Measurable business outcomes

## Website Hero Style

Recommended hero ingredients:

- NorthWing Midnight background
- Subtle data or workflow paths
- Connected business process nodes
- Dashboard UI fragments
- Mobile application elements
- Restrained AI references
- Flight Gold upward trajectory
- Clear negative space for headline and CTA

## Product Visualization Style

For dashboards, apps, and SaaS concepts:

- Use clean browser frames
- Use consistent device mockups
- Use subtle shadows
- Keep annotations minimal
- Use Flight Blue for active states
- Use Flight Gold for key outcomes or directional highlights

## Photography Direction

When original photography is used:

- Show real work environments
- Use clean, natural lighting
- Prefer candid, purposeful activity
- Avoid staged corporate stock-photo behavior
- Keep compositions simple and credible
